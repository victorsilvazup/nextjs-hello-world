# Next.js Hello World

Este é um projeto simples de "Hello World" criado com Next.js e Tailwind CSS.

## Pré-requisitos

- Node.js 14.6.0 ou superior
- npm (geralmente vem com Node.js)

## Como iniciar

1. Clone este repositório:
   ```
   git clone <URL_DO_REPOSITÓRIO>
   cd nextjs-hello-world
   ```

2. Instale as dependências:
   ```
   npm install
   ```

3. Execute o servidor de desenvolvimento:
   ```
   npm run dev
   ```

4. Abra [http://localhost:3000](http://localhost:3000) no seu navegador para ver o resultado.

## Scripts disponíveis

- `npm run dev`: Inicia o servidor de desenvolvimento.
- `npm run build`: Cria uma versão otimizada para produção.
- `npm start`: Inicia o servidor de produção.
- `npm run lint`: Executa o linter para verificar o código.

## Tecnologias utilizadas

- [Next.js](https://nextjs.org/)
- [React](https://reactjs.org/)
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS](https://tailwindcss.com/)

## Estrutura do projeto

- `app/`: Contém os componentes e páginas da aplicação.
- `public/`: Arquivos estáticos que serão servidos diretamente.
- `app/layout.tsx`: Layout principal da aplicação.
- `app/page.tsx`: Página inicial com a mensagem "Hello World".
